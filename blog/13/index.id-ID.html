Dev snapshot: Godot 3.2 beta 3

Banyak perbaikan yang telah dilakukan sejak versi beta kami sebelumnya, mencakup masalah rendering, masalah khusus porting terutama di iOS dan Windows, dan banyak perbaikan lainnya di seluruh bagian editor. Karena adanya masalah dalam proses build kami, rilis ini tidak menyertakan build Mono yang biasa, tetapi kami sedang bekerja keras untuk memperbaikinya dan menyediakan build Mono lagi dengan 3.2 beta 4.

Dev snapshot: Godot 3.2 beta 2

Kami sekarang merilis Godot 3.2 beta 2 dengan perbaikan bug selama dua minggu dari snapshot sebelumnya. Perubahan penting termasuk penambahan templat ekspor WebAssembly untuk build Mono, serta dukungan C# 8 melalui Mono 6.6.0 Preview.
